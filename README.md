# 📊 Retail Sales Performance Dashboard (Power BI)

## 🧾 Project Overview
This Power BI project analyzes retail sales data to identify key business insights such as sales performance, profit trends, category-wise performance, and regional distribution.

The dashboard helps in making data-driven business decisions by visualizing important KPIs and trends.

---

## 🎯 Objective
- Analyze overall sales and profit performance  
- Identify top-performing categories and regions  
- Track monthly sales trends  
- Improve business decision-making using data visualization  

---

## 🛠 Tools Used
- Power BI  
- Microsoft Excel (CSV Dataset)

---

## 📊 Dashboard Features
- KPI Cards (Total Sales, Total Profit, Total Orders)  
- Sales by Category  
- Sales by Region  
- Monthly Sales Trend Analysis  
- Interactive Filters (Slicers)  
- Sales Forecasting  

---

## 🔍 Key Insights
- Technology category shows highest sales  
- Certain categories have high sales but low profit  
- Regional performance varies significantly  
- Sales follow seasonal patterns  
- Forecasting helps predict future trends
